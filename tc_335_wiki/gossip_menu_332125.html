The `gossip_menu` table

Description of the fields

entry

This must match the entry you added to creature_template. This also
groups the options from gossip_menu_option and displays all options associated with this ID.

Note: If adding your own custom menu options, then it is common practice to start with an ID at or above 50,000 just to be safe that it doesn't conflict with other gossip menu id's.

text_id

This links to the npc_text.ID for the gossip you want to be initially displayed. Also this tells the NPC what to say at the top of the options menu when it is displayed.
