Description of the fields

guid

The GUID of the character. See characters.guid

instance

The instance ID. See instance.id

permanent

Boolean 0 or 1 controlling if the player has been bound to the instance. A player is bound to the instance only when he (or his party/raid) kills a creature with the CREATURE_FLAG_EXTRA_INSTANCE_BIND flag set in the flags_extras field.
